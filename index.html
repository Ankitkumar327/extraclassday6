Explain Absolute and Relative Positons with examples and words?

Position relative: Position relative means that position based on browsers screen. Position relative mostly applied to parent-container.

Position absolute: Position absoute means that position will be based on it parent container position, if the child container doesnot have any parent-container then it will go for the root. position absolute mostly works with child container.

Explain the difference between fixed and sticky?

Position Fixed means that it will remain fixed to the browsers screen.

Position Sticky is the combination of relative and fixed, it means that when we apply position sticky to the box then when it reaches the parameters applied it will get fixed.
